# Open Creative Studio Website

A beautiful, fully self-contained community creative learning hub website for Open Creative Studio (OCS).

## About

Open Creative Studio is a community learning hub offering programs in the arts, design, music, culinary skills, movement, gaming, STEM, humanities and beyond — for every curious mind.

**The Open Ground** is Chapter I — a season of planting, building, and discovering what this community needs.

## Features

- **19 Program Categories** - Visual Arts, Design & Media, Studio Crafts, Music, Dance, Movement & Wellness, Self-Defense, Sports, Culinary Arts, Nature & Sustainability, Fashion Design, Life Skills, Gaming, STEM, Language Arts, Community Engagement, Humanities & Culture, Healthy Aging, and Youth & Kids programs

- **100+ Classes** - Browse, filter, and search across drop-in sessions, sign-up based classes, team-based activities, private lessons, and more

- **Multi-Level Content** - Programs for beginners, advancing students, and all-levels learners, including specialized tracks for seniors (55+) and youth (ages 3-18)

- **Events Calendar** - Upcoming workshops and a full archive of past events with recaps

- **Community Hub** - Values, announcements, and community engagement opportunities

- **Shop** - Merchandise, class passes, memberships, supplies, and digital resources

## Technology Stack

- **Single-file deployment** - All HTML, CSS, and JavaScript in one file
- **No external dependencies** - Uses only Google Fonts for typography
- **Fully responsive** - Mobile-first design with CSS Grid and Flexbox
- **Pure JavaScript** - Client-side filtering, navigation, and interactivity with no build tools required

## Font Families Used

- **Space Grotesk** - Headlines and UI elements
- **Cormorant Garamond** - Serif accents and italics
- **Inter** - Body text and descriptions

## Color Palette

A carefully curated palette of 19+ colors representing each program category, from terra and mustard to forest and teal.

## Getting Started

Simply clone or download this repository and open `index.html` in a web browser. No build process or server required.

## Future Integrations

The site is designed to be easily extended with:

- **Stripe** - Payment processing for class passes and memberships
- **Ticket Tailor** - Event registration and ticketing
- **Framer or Webflow** - Visual design system migration
